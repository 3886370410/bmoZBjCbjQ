# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行水细菌检测
- 提供7000张标注数据集，包含68类生物类别
- 支持选择不同模型进行检测
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择是否显示标签和原图

## 适用场景
- 水质监测与分析
- 微生物研究
- 环境保护
- 生物多样性调查
- 水资源管理

## 技术特点
- 基于YOLOv8深度学习模型
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 支持Python 3.9及相关依赖库(opencv-python, PyQt5, ultralytics, torch1.9等)
- 提供完整UI界面和图标资源
- 包含训练好的模型及训练结果
- 配有详细的项目运行教程
